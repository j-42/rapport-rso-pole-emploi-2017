Comment lancer votre master offline

Sur Windows

  1. dezippez le fichier master.zip en double cliquant dessus
  2. ouvrir votre dossier dézippé
  3. lancer la lecture en double-cliquant sur le fichier "start.exe"

Sur Mac

  1. dezippez le fichier master.zip en double cliquant dessus
  2. dans votre dossier, dezippez le fichier start.zip en doublecliquant dessus
  3. (Sur Sierra)Double cliquez sur start.app pour lancer la publication. Si vous rencontrez une erreur, déplacer start.app sur votre bureau et remettez le ensuite dans son dossier.
  4. Exécuter l'application "start"


Comment installer votre master online

Pré-requis

  • Votre serveur Web (Apache, Nginx, Microsoft IIS, etc), doit etre configuré au minimum avec les Types MIME suivants:
    • JSON (application/json)
    • XML (application/xml)
    • HTML (text/html)
    • JavaScript (text/javascript)
    • CSS (text/css)
    • JPG (image/jpeg)
    • PNG (image/png)
    • SVG (image/svg+xml)
    • SWF (application/x-shockwave-flash)
    • PDF (application/pdf)
    • MP4 (video/mp4)
  • Autorisez à servir du contenu depuis les dossiers suivants:
    • .drive
    • .gallery

Installation

  1. Dezippez le fichier master.zip en double cliquant dessus
  2. Déplacez le dossier créé dans l'espace Web prévu sur votre serveur
