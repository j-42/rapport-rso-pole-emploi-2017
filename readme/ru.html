Как запустить офлайн мастер

На Windows

  1. Распакуйте скачанный zip-архив в новую папку
  2. Откройте новую папку
  3. Запустите файл "start.exe"

На Mac

  1. Распакуйте скачанный zip-архив
  2. Распакуйте файл "start.zip" в ту же папку
  3. Двойной клик на start запустит публикацию. Если у вас возникла ошибка, то перетащите файл на рабочий стол, а затем верните его в исходную папку
  4. Запустите файл "start"


Как установить онлайн мастер

Требования

  • Веб-сервер (Apache, Nginx, Microsoft IIS, и т.д.), сконфигурированный как минимум для следующих MIME-типов (доступно по умолчанию для большинства современных версий большинства веб-серверов):
    • JSON (application/json)
    • XML (application/xml)
    • HTML (text/html)
    • JavaScript (text/javascript)
    • CSS (text/css)
    • JPG (image/jpeg)
    • PNG (image/png)
    • SVG (image/svg+xml)
    • SWF (application/x-shockwave-flash)
    • PDF (application/pdf)
    • MP4 (video/mp4)
  • Разрешение на показ содержимого из следующего набора префиксных папок:
    • .drive
    • .gallery

Установка

  1. Распакуйте скачанный zip-архив в новую папку
  2. Переместите новую папку в нужное место внутри созданной папки на веб-сервере
