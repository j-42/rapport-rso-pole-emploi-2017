Como ejecutar tu máster offline

En Windows

  1. Clicka en el archivo master.zip y descomprímelo
  2. Abre la carpeta resultante
  3. Clicka en el archivo "start.exe"

En Mac

  1. Clicka en el archivo master.zip y descomprímelo
  2. Abre la carpeta resultante y descomprime el archivo "start.zip"
  3. (En Sierra) Clicka en "start.app" para lanzar la publicación. Si no funciona, arrastra el start.app al escritorio, y luego vuelve a arrastrarlo de nuevo a la carpeta dónde lo descomprimiste.
  4. Ejecutar el archivo "start"


Como instalar tu master online

Requisitos

  • Tu servidor (Apache, Nginx, Microsoft IIS, etc) debe estar configurado al menos con los siguientes Types MIME (disponibles en la mayoría de servidores recientes):
    • JSON (application/json)
    • XML (application/xml)
    • HTML (text/html)
    • JavaScript (text/javascript)
    • CSS (text/css)
    • JPG (image/jpeg)
    • PNG (image/png)
    • SVG (image/svg+xml)
    • SWF (application/x-shockwave-flash)
    • PDF (application/pdf)
    • MP4 (video/mp4)
  • Debes autorizar al servidor a lanzar contenido desde las carpetas siguientes:
    • .drive
    • .gallery

Instalación

  1. Clicka en el archivo master.zip y descomprímelo
  2. Mueve el contenido de la carpeta resultante en el directorio de tu servidor que quieras
